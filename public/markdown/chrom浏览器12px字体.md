### 1.chrome限制12px以下的 依旧12px
``` bash
-webkit-text-size-adjust:none; //以前版本chrome适用，现在只有手机端可以用了
```
最新版本chrome要用新的办法了：
```bash
font-size: 12px;
transform: scale(0.8); /*2D缩放*/
display: inline-block;
```
