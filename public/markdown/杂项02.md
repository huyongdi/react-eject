### vue-init项目 webpack打包之后图片等路径会有问题

在utils的js里面修改插件:加注释的这句
```bash
if (options.extract) {
      return ExtractTextPlugin.extract({
        use: loaders,
        fallback: 'vue-style-loader',
        publicPath:'../../' /*处理打包后路径错误*/
      })
    } else {
      return ['vue-style-loader'].concat(loaders)
    }
```
